# M.Tech Labs Repository

Welcome to my **M.Tech Labs** repository! 

This repository contains all the lab assignments, projects, and related codebase developed during my M.Tech journey. It is structured to maintain all coursework organized by subject. 

## 📂 Repository Structure

The folders in this repository correspond to various lab courses I've undertaken. Here is an overview of the modules included:

- **[`ComputerSystemDesignLab/`](./ComputerSystemDesignLab/)** - Assignments and code related to Computer System Design.
- **[`DSA/`](./DSA/)** - Data Structures and Algorithms implementations, assignments, and problem-solving exercises.
- **[`DS_AI/`](./DS_AI/)** - Data Science and Artificial Intelligence lab works, including exploratory data analysis and modeling.
- **[`Drone_Airsim/`](./Drone_Airsim/)** - Projects and simulations involving Drones and Microsoft AirSim.
- **[`MLDL/`](./MLDL/)** - Machine Learning and Deep Learning labs, covering models from basic regressions to complex neural networks.
- **[`Networks/`](./Networks/)** - Computer Networks assignments involving socket programming, protocols, and network simulations.
- **[`WebDev/`](./WebDev/)** - Web Development projects, including frontend and backend technologies, managing state, databases, and full-stack applications.
- **[`other/`](./other/)** - Miscellaneous tasks, utilities, and helper scripts used throughout the coursework.

## 🛠️ Tools & Technologies
Across these labs, a variety of technologies, languages, and tools have been utilized:
- **Programming Languages:** C++, Python, JavaScript, Java, PHP, etc.
- **Libraries & Frameworks:** React, PyTorch, TensorFlow, etc. (varies by course).
- **Other Utilities:** AirSim, Apache Tomcat, SQLite, and standard Data Science toolkits (Pandas, NumPy, Scikit-learn). 

## 📝 Navigation
Each directory contains code related directly to corresponding labs. There could be further `README.md` files placed in individual lab folders describing specific exercises and execution steps.
